# Cours de cartographie Web pour les Licence Pro de l'ENSG, groupe initiation

## Référence cours magistral
Le cours magistral s'appuie sur le super cours de Vincent De Olivera [@iamvdo](https://github.com/iamvdo), le web de A à Z : [https://web.iamvdo.me/](https://web.iamvdo.me/).

Jour 1 :

- HTML
- CSS
  + Langage
  + Sélecteurs
  + Propriétés

Jour 2 :

- CSS
  + Positionnement
  + Responsive Web Design
  + Outils
- Javascript
  + Principe
  + Langage

Jour 3 :

- Javascript
  + Langage avancé
    * DOM
    * API Fetch
- Webmapping

Jour 4 :

- Finalisation du projet


## Projet

Le cours s'organise autour d'un projet de site web, sur lequel vous serez évalués. Il est conseillé d'avancer dessus au fur et à mesure du cours en voyant les différents sujets.
